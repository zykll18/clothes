# Phase 1 Foundation Alignment Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Align the repository around the real Next.js 16 + React 19 + Prisma + SQLite + DashScope stack, remove misleading structure drift, and get `npm run lint` back to a passing state without expanding into feature work.

**Architecture:** Treat `prisma/schema.prisma` and the API routes as the persistence truth, keep preview-only transform state out of shared entity types, and use lint-driven red/green cycles to fix the current engineering blockers. Keep the UI structure intact except for obviously invalid metadata and navigation residues.

**Tech Stack:** Next.js 16, React 19, TypeScript 5, Tailwind CSS 4, Prisma 6, SQLite, DashScope, ESLint 9

---
